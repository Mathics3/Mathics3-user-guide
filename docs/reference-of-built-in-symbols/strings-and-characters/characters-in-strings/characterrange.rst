CharacterRange
==============

`WMA link <https://reference.wolfram.com/language/ref/CharacterRange.html>`_


:code:`CharacterRange` [":math:`a`", ":math:`b`"]
    returns a list of the Unicode characters from :math:`a` to :math:`b` inclusive.





>>> CharacterRange["a", "e"]
  = {a, b, c, d, e}
>>> CharacterRange["b", "a"]
  = {}
