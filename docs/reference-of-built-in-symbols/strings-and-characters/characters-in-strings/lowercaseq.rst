LowerCaseQ
==========

`WMA link <https://reference.wolfram.com/language/ref/LowerCaseQ.html>`_


:code:`LowerCaseQ` [:math:`s`]
    returns True if :math:`s` consists wholly of lower case characters.





>>> LowerCaseQ["abc"]
    =

:math:`\text{True}`



An empty string returns True.

>>> LowerCaseQ[""]
    =

:math:`\text{True}`
