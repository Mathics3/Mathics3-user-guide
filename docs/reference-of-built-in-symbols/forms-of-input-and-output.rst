Forms of Input and Output
=========================

A *Form* format specifies the way Mathics Expression input is read or output written.

The variable `$OutputForms' </doc/reference-of-built-in-symbols/forms-of-input-and-output/form-variables/$outputforms/>`_ has a list of Forms defined.

See also `WMA link <https://reference.wolfram.com/language/tutorial/TextualInputAndOutput.html#12368>`_.
