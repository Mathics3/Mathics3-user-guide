Mod
===

`WMA link <https://reference.wolfram.com/language/ref/Mod.html>`_


:code:`Mod` [:math:`x`, :math:`m`]
    returns :math:`x` modulo :math:`m`.





>>> Mod[14, 6]

    =
:math:`2`


>>> Mod[-3, 4]

    =
:math:`1`


>>> Mod[-3, -4]

    =
:math:`-3`


>>> Mod[5, 0]

    Mod::divz The argument 0 should be nonzero.

    =
:math:`\text{Mod}\left[5,0\right]`
