ConstantArray
=============

`WMA link <https://reference.wolfram.com/language/ref/ConstantArray.html>`_


:code:`ConstantArray` [:math:`expr`, :math:`n`]
    returns a list of :math:`n` copies of :math:`expr`.





>>> ConstantArray[a, 3]
  = {a, a, a}
>>> ConstantArray[a, {2, 3}]
  = {{a, a, a}, {a, a, a}}
