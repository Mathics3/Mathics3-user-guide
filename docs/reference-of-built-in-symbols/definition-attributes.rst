Definition Attributes
=====================

While a definition like :code:`cube[x_] = x^3`  gives a way to specify `<values>`_ of a function, `<attributes>`_ allow a way to specify general properties of functions and symbols. This is independent of the parameters they take and the values they produce.

The builtin-attributes having a predefined meaning in \Mathics which are described below.

However in contrast to \Mathematica, you can set any symbol as an attribute.
