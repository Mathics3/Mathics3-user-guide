BesselJZero
===========

`WMA link <https://reference.wolfram.com/language/ref/BesselJZero.html>`_


:code:`BesselJZero` [:math:`n`, :math:`k`]
    returns the :math:`k`th zero of the Bessel function of the first kind :math:`J_n(z)`.





>>> N[BesselJZero[0, 1]]

    =
:math:`2.40483`


>>> N[BesselJZero[0, 1], 10]

    =
:math:`2.404825558`
