StruveL
=======

`Modified Struve functions L <https://en.wikipedia.org/wiki/Struve_function>`_

:code:`StruveL` [:math:`n`, :math:`z`]
    returns the modified Struve function :math:`L_n(z)`.





>>> StruveL[1.5, 3.5]

    =
:math:`4.41126`


>>> Plot[StruveL[0, x], {x, 0, 5}]

    =
