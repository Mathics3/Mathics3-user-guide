AiryBiPrime
===========

`WMA link <https://reference.wolfram.com/language/ref/AiryBiPrime.html>`_


:code:`AiryBiPrime` [:math:`x`]
    returns the derivative of the Airy function of the second
    kind :code:`AiryBi` [:math:`x`].





Exact values:

>>> AiryBiPrime[0]
  = 3 ^ (1 / 6) / Gamma[1 / 3]

Numeric evaluation:

>>> AiryBiPrime[0.5]
  = 0.544573
