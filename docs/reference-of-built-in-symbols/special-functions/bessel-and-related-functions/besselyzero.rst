BesselYZero
===========

`WMA link <https://reference.wolfram.com/language/ref/BesselYZero.html>`_


:code:`BesselYZero` [:math:`n`, :math:`k`]
    returns the :math:`k`th zero of the Bessel function of the second kind :math:`Y_n(z)`.





>>> N[BesselYZero[0, 1]]
  = 0.893577
>>> N[BesselYZero[0, 1], 10]
  = 0.8935769663
