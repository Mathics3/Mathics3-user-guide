Zeta
====

`Riemenn zeta function <https://en.wikipedia.org/wiki/Riemann_zeta_function>`_ (`WMA <https://reference.wolfram.com/language/ref/Zeta.html>`_)


:code:`Zeta` [:math:`z`]
    returns the Riemann zeta function of :math:`z`.





>>> Zeta[2]

    =
:math:`\frac{ \pi ^2}{6}`


>>> Zeta[-2.5 + I]

    =
:math:`0.0235936+0.0014078 I`


>>> Plot[Zeta[z], {z, -20, 10}]
    = -Graphics-`
