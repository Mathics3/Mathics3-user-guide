PolyLog
=======

`Polylogarithm <https://en.wikipedia.org/wiki/Polylogarithm>`_ (`WMA <https://reference.wolfram.com/language/ref/PolyLog.html>`_)


:code:`PolyLog` [:math:`n`, :math:`z`]
    returns the polylogarithm function :math:`Li_n(z)`.





>>> PolyLog[s, 1]
  = Zeta[s]
>>> PolyLog[-7, I] //Chop
  = 136.
