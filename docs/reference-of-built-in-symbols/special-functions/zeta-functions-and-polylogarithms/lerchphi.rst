LerchPhi
========

`Lerch transcendent <https://en.wikipedia.org/wiki/Lerch_transcendent>`_ (`WMA <https://reference.wolfram.com/language/ref/LerchPhi.html>`_)


:code:`LerchPhi[z,s,a]`
    gives the Lerch transcendent :math:`\Phi(z,s,a)`.





>>> LerchPhi[2, 3, -1.5]
  = 19.3893 - 2.1346 I
>>> LerchPhi[1, 2, 1/4]
  = 17.1973
