Begin
=====

`WMA link <https://reference.wolfram.com/language/ref/Begin.html>`_


:code:`Begin` [:math:`context`]
    temporarily sets the current context to :math:`context`.





>>> Begin["test`"]

    =
:math:`\text{test\`{}}`


>>> End[]

    =
:math:`\text{test\`{}}`


>>> End[]

    End::noctx No previous context defined.

    =
:math:`\text{Global\`{}}`
