Null
====

`WMA link <https://reference.wolfram.com/language/ref/Null.html>`_


:code:`Null`
    is the implicit result of expressions that do not yield a result.





>>> FullForm[a:=b]

    =
:math:`\text{Null}`



It is not displayed in StandardForm,

>>> a:=b



in contrast to the empty string:

>>> ""

    =
:math:`\text{}`
