MatrixForm
==========

`WMA link <https://reference.wolfram.com/language/ref/MatrixForm.html>`_


:code:`MatrixForm` [:math:`m`]
    displays a matrix :math:`m`, hiding the underlying list structure.





>>> Array[a,{4,3}]//MatrixForm
  = a[1, 1]   a[1, 2]   a[1, 3]
    
    a[2, 1]   a[2, 2]   a[2, 3]
    
    a[3, 1]   a[3, 2]   a[3, 3]
    
    a[4, 1]   a[4, 2]   a[4, 3]
