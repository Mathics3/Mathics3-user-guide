PrimePi
=======

`Prime numbers <https://reference.wolfram.com/language/ref/PrimePi.html>`_


:code:`PrimePi` [:math:`x`]
    gives the number of primes less than or equal to :math:`x`.





PrimePi is the inverse of Prime:

>>> PrimePi[2]
    =

:math:`1`


>>> PrimePi[100]
    =

:math:`25`


>>> PrimePi[-1]
    =

:math:`0`


>>> PrimePi[3.5]
    =

:math:`2`


>>> PrimePi[E]
    =

:math:`1`
