Ceiling
=======

`WMA link <https://reference.wolfram.com/language/ref/Ceiling.html>`_


:code:`Ceiling` [:math:`x`]
    gives the smallest integer greater than or equal to :math:`x`.





>>> Ceiling[1.2]
  = 2
>>> Ceiling[3/2]
  = 2

For complex :math:`x`, take the ceiling of real an imaginary parts.

>>> Ceiling[1.3 + 0.7 I]
  = 2 + I
