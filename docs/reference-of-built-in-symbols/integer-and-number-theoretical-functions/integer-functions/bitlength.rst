BitLength
=========

`WMA link <https://reference.wolfram.com/language/ref/BitLength.html>`_


:code:`BitLength` [:math:`x`]
    gives the number of bits needed to represent the integer :math:`x`. :math:`x`'s sign is ignored.





>>> BitLength[1023]
    =

:math:`10`


>>> BitLength[100]
    =

:math:`7`


>>> BitLength[-5]
    =

:math:`3`


>>> BitLength[0]
    =

:math:`0`
