InverseHaversine
================

`WMA link <https://reference.wolfram.com/language/ref/InverseHaversine.html>`_


:code:`InverseHaversine` [:math:`z`]
    returns the inverse haversine function of :math:`z`.





>>> InverseHaversine[0.5]
  = 1.5708
>>> InverseHaversine[1 + 2.5 I]
  = 1.76459 + 2.33097 I
