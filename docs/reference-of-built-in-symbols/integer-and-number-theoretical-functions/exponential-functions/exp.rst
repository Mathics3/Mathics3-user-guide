Exp
===

`WMA link <https://reference.wolfram.com/language/ref/Exp.html>`_


:code:`Exp` [:math:`z`]
    returns the exponential function of :math:`z`.





>>> Exp[1]

    =
:math:`E`


>>> Exp[10.0]

    =
:math:`22026.5`


>>> Exp[x] //FullForm

    =
:math:`\text{Power}\left[E, x\right]`
