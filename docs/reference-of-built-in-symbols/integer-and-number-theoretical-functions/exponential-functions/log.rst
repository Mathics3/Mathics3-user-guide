Log
===

`WMA link <https://reference.wolfram.com/language/ref/Log.html>`_


:code:`Log` [:math:`z`]
    returns the natural logarithm of :math:`z`.





>>> Log[{0, 1, E, E * E, E ^ 3, E ^ x}]

    =
:math:`\left\{-\infty ,0,1,2,3,\text{Log}\left[E^x\right]\right\}`


>>> Log[0.]

    =
:math:`\text{Indeterminate}`
