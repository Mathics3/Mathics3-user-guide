Log10
=====

`WMA link <https://reference.wolfram.com/language/ref/Log10.html>`_


:code:`Log10` [:math:`z`]
    returns the base-10 logarithm of :math:`z`.





>>> Log10[1000]
    =

:math:`3`


>>> Log10[{2., 5.}]
    =

:math:`\left\{0.30103,0.69897\right\}`


>>> Log10[E ^ 3]
    =

:math:`\frac{3}{\text{Log}\left[10\right]}`
