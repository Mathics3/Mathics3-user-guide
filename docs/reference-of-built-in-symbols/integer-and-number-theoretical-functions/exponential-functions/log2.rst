Log2
====

`WMA link <https://reference.wolfram.com/language/ref/Log2.html>`_


:code:`Log2` [:math:`z`]
    returns the base-2 logarithm of :math:`z`.





>>> Log2[4 ^ 8]
    =

:math:`16`


>>> Log2[5.6]
    =

:math:`2.48543`


>>> Log2[E ^ 2]
    =

:math:`\frac{2}{\text{Log}\left[2\right]}`
