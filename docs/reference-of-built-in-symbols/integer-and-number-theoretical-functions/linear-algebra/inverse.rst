Inverse
=======

`WMA link <https://reference.wolfram.com/language/ref/Inverse.html>`_


:code:`Inverse` [:math:`m`]
    computes the inverse of the matrix :math:`m`.





>>> Inverse[{{1, 2, 0}, {2, 3, 0}, {3, 4, 1}}]
  = {{-3, 2, 0}, {2, -1, 0}, {1, -2, 1}}
>>> Inverse[{{1, 0}, {0, 0}}]
  = Inverse[{{1, 0}, {0, 0}}]
