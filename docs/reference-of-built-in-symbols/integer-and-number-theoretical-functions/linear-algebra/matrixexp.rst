MatrixExp
=========

`WMA link <https://reference.wolfram.com/language/ref/MatrixExp.html>`_


:code:`MatrixExp` [:math:`m`]
    computes the exponential of the matrix :math:`m`.





>>> MatrixExp[{{0, 2}, {0, 1}}]
  = {{1, -2 + 2 E}, {0, E}}
>>> MatrixExp[{{1.5, 0.5}, {0.5, 2.0}}]
  = {{5.16266, 3.02952}, {3.02952, 8.19218}}
