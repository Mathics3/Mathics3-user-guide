MatrixPower
===========

`WMA link <https://reference.wolfram.com/language/ref/MatrixPower.html>`_


:code:`MatrixPower` [:math:`m`, :math:`n`]
    computes the :math:`n`th power of a matrix :math:`m`.





>>> MatrixPower[{{1, 2}, {1, 1}}, 10]
  = {{3363, 4756}, {2378, 3363}}
>>> MatrixPower[{{1, 2}, {2, 5}}, -3]
  = {{169, -70}, {-70, 29}}
