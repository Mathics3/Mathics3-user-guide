Denominator
===========

`WMA link <https://reference.wolfram.com/language/ref/Denominator.html>`_


:code:`Denominator` [:math:`expr`]
    gives the denominator in :math:`expr`.





>>> Denominator[a / b]

    =
:math:`b`


>>> Denominator[2 / 3]

    =
:math:`3`


>>> Denominator[a + b]

    =
:math:`1`
