ArcTanh
=======

`WMA link <https://reference.wolfram.com/language/ref/ArcTanh.html>`_


:code:`ArcTanh` [:math:`z`]
    returns the inverse hyperbolic tangent of :math:`z`.





>>> ArcTanh[0]
  = 0
>>> ArcTanh[1]
  = Infinity
>>> ArcTanh[0]
  = 0
>>> ArcTanh[.5 + 2 I]
  = 0.0964156 + 1.12656 I
>>> ArcTanh[2 + I]
  = ArcTanh[2 + I]
