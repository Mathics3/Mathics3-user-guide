Show
====

`WMA link <https://reference.wolfram.com/language/ref/Show.html>`_


:code:`Show` [:math:`graphics`, :math:`options`]
    shows a list of graphics with the specified options added.





>>> Show[{Plot[x, {x, 0, 10}], ListPlot[{1,2,3}]}]
