PlotPoints
==========

`WMA link <https://reference.wolfram.com/language/ref/PlotPoints.html>`_


:code:`PlotPoints :math:`n``
    A number specifies how many initial sample points to use.





>>> Plot[Sin[Cos[x^2]],{x,-4,4}, PlotPoints->22]
