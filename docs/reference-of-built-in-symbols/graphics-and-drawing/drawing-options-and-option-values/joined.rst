Joined
======

`WMA link <https://reference.wolfram.com/language/ref/Joined.html>`_


:code:`Joined :math:`boolean``
    is an option for :code:`Plot`  that gives whether to join points to make lines.





>>> ListPlot[Table[n ^ 2, {n, 10}], Joined->True]
