UpValue-related assignments
===========================

An *UpValue* is a definition associated with a symbols that does not appear directly its head.

See `Associating Definitions with Different Symbols <https://reference.wolfram.com/language/tutorial/TransformationRulesAndDefinitions.html#6972>`_.
