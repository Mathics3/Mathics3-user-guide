VectorAngle
===========

`WMA link <https://reference.wolfram.com/language/ref/VectorAngle.html>`_


:code:`VectorAngle` [:math:`u`, :math:`v`]
    gives the angles between vectors :math:`u` and :math:`v`





>>> VectorAngle[{1, 0}, {0, 1}]
    =

:math:`\frac{ \pi }{2}`


>>> VectorAngle[{1, 2}, {3, 1}]
    =

:math:`\frac{ \pi }{4}`


>>> VectorAngle[{1, 1, 0}, {1, 0, 1}]
    =

:math:`\frac{ \pi }{3}`
