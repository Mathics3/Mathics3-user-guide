ColorQuantize
=============

`WMA link <https://reference.wolfram.com/language/ref/ColorQuantize.html>`_


:code:`ColorQuantize` [:math:`image`, :math:`n`]
    gives a version of :math:`image` using only :math:`n` colors.





>>> img = Import["ExampleData/hedy.tif"];

>>> ColorQuantize[img, 6]
  = -Image-
