EdgeDetect
==========

`WMA link <https://reference.wolfram.com/language/ref/EdgeDetect.html>`_


:code:`EdgeDetect` [:math:`image`]
    returns an image showing the edges in :math:`image`.





>>> hedy = Import["ExampleData/hedy.tif"];

>>> EdgeDetect[hedy]
  = -Image-
>>> EdgeDetect[hedy, 5]
  = -Image-
>>> EdgeDetect[hedy, 4, 0.5]
  = -Image-
