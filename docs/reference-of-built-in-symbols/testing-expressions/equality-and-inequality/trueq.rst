TrueQ
=====

`WMA link <https://reference.wolfram.com/language/ref/TrueQ.html>`_


:code:`TrueQ` [:math:`expr`]
    returns :code:`True`  if and only if :math:`expr` is :code:`True` .





>>> TrueQ[True]
    =

:math:`\text{True}`


>>> TrueQ[False]
    =

:math:`\text{False}`


>>> TrueQ[a]
    =

:math:`\text{False}`
