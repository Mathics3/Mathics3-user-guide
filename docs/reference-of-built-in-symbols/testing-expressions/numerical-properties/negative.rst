Negative
========

`WMA link <https://reference.wolfram.com/language/ref/Negative.html>`_


:code:`Negative` [:math:`x`]
    returns :code:`True`  if :math:`x` is a negative real number.





>>> Negative[0]
  = False
>>> Negative[-3]
  = True
>>> Negative[10/7]
  = False
>>> Negative[1+2I]
  = False
>>> Negative[a + b]
  = Negative[a + b]
