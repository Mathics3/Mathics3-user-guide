Negative
========

`WMA link <https://reference.wolfram.com/language/ref/Negative.html>`_


:code:`Negative` [:math:`x`]
    returns :code:`True`  if :math:`x` is a negative real number.





>>> Negative[0]
    =

:math:`\text{False}`


>>> Negative[-3]
    =

:math:`\text{True}`


>>> Negative[10/7]
    =

:math:`\text{False}`


>>> Negative[1+2I]
    =

:math:`\text{False}`


>>> Negative[a + b]
    =

:math:`\text{Negative}\left[a+b\right]`
