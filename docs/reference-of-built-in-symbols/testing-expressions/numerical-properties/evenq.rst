EvenQ
=====

`WMA link <https://reference.wolfram.com/language/ref/EvenQ.html>`_


:code:`EvenQ` [:math:`x`]
    returns :code:`True`  if :math:`x` is even, and :code:`False`  otherwise.





>>> EvenQ[4]
    =

:math:`\text{True}`


>>> EvenQ[-3]
    =

:math:`\text{False}`


>>> EvenQ[n]
    =

:math:`\text{False}`
