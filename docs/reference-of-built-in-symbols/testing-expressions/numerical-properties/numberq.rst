NumberQ
=======

`WMA link <https://reference.wolfram.com/language/ref/NumberQ.html>`_


:code:`NumberQ` [:math:`expr`]
    returns :code:`True`  if :math:`expr` is an explicit number, and :code:`False`            otherwise.





>>> NumberQ[3+I]
    =

:math:`\text{True}`


>>> NumberQ[5!]
    =

:math:`\text{True}`


>>> NumberQ[Pi]
    =

:math:`\text{False}`
