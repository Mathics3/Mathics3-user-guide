Not
===

`WMA link <https://reference.wolfram.com/language/ref/Not.html>`_


:code:`Not` [:math:`expr`]
    same as

:code:`!:math:`expr``
    negates the logical expression :math:`expr`.





>>> !True

    =
:math:`\text{False}`


>>> !False

    =
:math:`\text{True}`


>>> !b

    =
:math:`!b`
