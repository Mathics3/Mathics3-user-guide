OrderedQ
========

`WMA link <https://reference.wolfram.com/language/ref/OrderedQ.html>`_


:code:`OrderedQ` [{:math:`a`, :math:`b`}]
    is :code:`True`  if :math:`a` sorts before :math:`b` according to canonical
    ordering.





>>> OrderedQ[{a, b}]
    =

:math:`\text{True}`


>>> OrderedQ[{b, a}]
    =

:math:`\text{False}`
