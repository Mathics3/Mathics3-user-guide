ListQ
=====

`WMA link <https://reference.wolfram.com/language/ref/ListQ.html>`_


:code:`ListQ` [:math:`expr`]
    tests whether :math:`expr` is a :code:`List` .





>>> ListQ[{1, 2, 3}]
    =

:math:`\text{True}`


>>> ListQ[{{1, 2}, {3, 4}}]
    =

:math:`\text{True}`


>>> ListQ[x]
    =

:math:`\text{False}`
