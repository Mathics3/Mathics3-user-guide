Angular Momentum
================

`Angular momentum <https://en.wikipedia.org/wiki/Angular_momentum>`_ in physics is the rotational analog of linear momentum. It is an important quantity in physics because it is a conserved quantity the total angular momentum of a closed system remains constant.
