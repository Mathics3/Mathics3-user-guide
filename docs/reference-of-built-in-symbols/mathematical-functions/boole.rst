Boole
=====

`WMA link <https://reference.wolfram.com/language/ref/Boole.html>`_


:code:`Boole[expr]`
    returns 1 if expr is True and 0 if expr is False.





>>> Boole[2 == 2]

    =
:math:`1`


>>> Boole[7 < 5]

    =
:math:`0`


>>> Boole[a == 7]

    =
:math:`\text{Boole}\left[a\text{==}7\right]`
