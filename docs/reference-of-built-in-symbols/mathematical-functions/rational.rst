Rational
========

`WMA link <https://reference.wolfram.com/language/ref/Rational.html>`_


:code:`Rational`
    is the head of rational numbers.

:code:`Rational` [:math:`a`, :math:`b`]
    constructs the rational number :math:`a` / :math:`b`.





>>> Head[1/2]

    =
:math:`\text{Rational}`


>>> Rational[1, 2]

    =
:math:`\frac{1}{2}`
