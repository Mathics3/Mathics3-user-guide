CapitalDifferentialD
====================

`WML link <https://reference.wolfram.com/language/ref/CapitalDifferentialD.html>`_


:code:`CapitalDifferentialD[:math:`x`]`
    displays ⅅ :math:`x`





>>> CapitalDifferentialD[x]

    =
:math:`\mathbb{D} x`


>>> \[CapitalDifferentialD]x

    =
:math:`\text{ⅅx}`
