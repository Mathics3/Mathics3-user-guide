GreaterGreater
==============

`WML link <https://reference.wolfram.com/language/ref/GreaterGreater.html>`_


:code:`GreaterGreater[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≫ :math:`y` ≫ ...





>>> GreaterGreater[x, y, z]

    =
:math:`x \gg y \gg z`


>>> a \[GreaterGreater] b

    =
:math:`a \gg b`
