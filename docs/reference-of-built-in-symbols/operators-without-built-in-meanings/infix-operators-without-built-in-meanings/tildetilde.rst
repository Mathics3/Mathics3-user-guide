TildeTilde
==========

`WML link <https://reference.wolfram.com/language/ref/TildeTilde.html>`_


:code:`TildeTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≈ :math:`y` ≈ ...





>>> TildeTilde[x, y, z]
    =

:math:`x \approx y \approx z`


>>> a \[TildeTilde] b
    =

:math:`a \approx b`
