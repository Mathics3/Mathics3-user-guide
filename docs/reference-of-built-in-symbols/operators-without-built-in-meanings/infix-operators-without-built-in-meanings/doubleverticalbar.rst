DoubleVerticalBar
=================

`WML link <https://reference.wolfram.com/language/ref/DoubleVerticalBar.html>`_


:code:`DoubleVerticalBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ∥ :math:`y` ∥ ...





>>> DoubleVerticalBar[x, y, z]
  = x ∥ y ∥ z
>>> a \[DoubleVerticalBar] b
  = a ∥ b
