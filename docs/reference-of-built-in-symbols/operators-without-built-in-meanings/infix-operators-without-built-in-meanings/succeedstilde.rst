SucceedsTilde
=============

`WML link <https://reference.wolfram.com/language/ref/SucceedsTilde.html>`_


:code:`SucceedsTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≿ :math:`y` ≿ ...





>>> SucceedsTilde[x, y, z]
    =

:math:`x \succsim y \succsim z`


>>> a \[SucceedsTilde] b
    =

:math:`a \succsim b`
