VerticalBar
===========

`WML link <https://reference.wolfram.com/language/ref/VerticalBar.html>`_


:code:`VerticalBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ❘ :math:`y` ❘ ...





>>> VerticalBar[x, y, z]
    =

:math:`x \shortmid y \shortmid z`


>>> a \[VerticalBar] b
    =

:math:`a \shortmid b`
