NotGreaterGreater
=================

`WML link <https://reference.wolfram.com/language/ref/NotGreaterGreater.html>`_


:code:`NotGreaterGreater[:math:`x`, :math:`y`, ...]`
    displays :math:`x`  :math:`y`  ...





>>> NotGreaterGreater[x, y, z]
  = x  y  z
>>> a \[NotGreaterGreater] b
  = a  b
