NotTildeTilde
=============

`WML link <https://reference.wolfram.com/language/ref/NotTildeTilde.html>`_


:code:`NotTildeTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≉ :math:`y` ≉ ...





>>> NotTildeTilde[x, y, z]

    =
:math:`x \not{\approx} y \not{\approx} z`


>>> a \[NotTildeTilde] b

    =
:math:`a \not{\approx} b`
