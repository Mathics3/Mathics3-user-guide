NotGreaterTilde
===============

`WML link <https://reference.wolfram.com/language/ref/NotGreaterTilde.html>`_


:code:`NotGreaterTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≵ :math:`y` ≵ ...





>>> NotGreaterTilde[x, y, z]
    =

:math:`x \not{\gtrsim} y \not{\gtrsim} z`


>>> a \[NotGreaterTilde] b
    =

:math:`a \not{\gtrsim} b`
