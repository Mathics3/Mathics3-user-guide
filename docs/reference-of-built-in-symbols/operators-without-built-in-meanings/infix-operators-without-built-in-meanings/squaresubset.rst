SquareSubset
============

`WML link <https://reference.wolfram.com/language/ref/SquareSubset.html>`_


:code:`SquareSubset[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊏ :math:`y` ⊏ ...





>>> SquareSubset[x, y, z]
    =

:math:`x \sqsubset y \sqsubset z`


>>> a \[SquareSubset] b
    =

:math:`a \sqsubset b`
