NotSucceeds
===========

`WML link <https://reference.wolfram.com/language/ref/NotSucceeds.html>`_


:code:`NotSucceeds[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊁ :math:`y` ⊁ ...





>>> NotSucceeds[x, y, z]

    =
:math:`x \nsucc y \nsucc z`


>>> a \[NotSucceeds] b

    =
:math:`a \nsucc b`
