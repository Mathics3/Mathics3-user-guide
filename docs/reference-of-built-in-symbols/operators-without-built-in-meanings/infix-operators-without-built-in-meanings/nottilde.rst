NotTilde
========

`WML link <https://reference.wolfram.com/language/ref/NotTilde.html>`_


:code:`NotTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≁ :math:`y` ≁ ...





>>> NotTilde[x, y, z]

    =
:math:`x \not{\sim} y \not{\sim} z`


>>> a \[NotTilde] b

    =
:math:`a \not{\sim} b`
