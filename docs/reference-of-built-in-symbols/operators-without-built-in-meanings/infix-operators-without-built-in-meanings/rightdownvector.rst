RightDownVector
===============

`WML link <https://reference.wolfram.com/language/ref/RightDownVector.html>`_


:code:`RightDownVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇂ :math:`y` ⇂ ...





>>> RightDownVector[x, y, z]

    =
:math:`x \underline{\downharpoonright} y \underline{\downharpoonright} z`


>>> a \[RightDownVector] b

    =
:math:`a \underline{\downharpoonright} b`
