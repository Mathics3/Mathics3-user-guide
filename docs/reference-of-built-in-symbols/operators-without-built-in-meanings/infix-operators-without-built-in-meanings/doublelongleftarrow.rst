DoubleLongLeftArrow
===================

`WML link <https://reference.wolfram.com/language/ref/DoubleLongLeftArrow.html>`_


:code:`DoubleLongLeftArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⟸ :math:`y` ⟸ ...





>>> DoubleLongLeftArrow[x, y, z]
  = x ⟸ y ⟸ z
>>> a \[DoubleLongLeftArrow] b
  = a ⟸ b
