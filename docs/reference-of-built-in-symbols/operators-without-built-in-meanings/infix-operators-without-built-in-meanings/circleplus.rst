CirclePlus
==========

`WML link <https://reference.wolfram.com/language/ref/CirclePlus.html>`_


:code:`CirclePlus[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊕ :math:`y` ⊕ ...





>>> CirclePlus[x, y, z]
    =

:math:`x  \oplus y  \oplus z`


>>> a \[CirclePlus] b
    =

:math:`a  \oplus b`
