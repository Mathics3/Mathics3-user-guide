LeftTriangle
============

`WML link <https://reference.wolfram.com/language/ref/LeftTriangle.html>`_


:code:`LeftTriangle[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊲ :math:`y` ⊲ ...





>>> LeftTriangle[x, y, z]

    =
:math:`x \triangleleft y \triangleleft z`


>>> a \[LeftTriangle] b

    =
:math:`a \triangleleft b`
