PrecedesSlantEqual
==================

`WML link <https://reference.wolfram.com/language/ref/PrecedesSlantEqual.html>`_


:code:`PrecedesSlantEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≼ :math:`y` ≼ ...





>>> PrecedesSlantEqual[x, y, z]
  = x ≼ y ≼ z
>>> a \[PrecedesSlantEqual] b
  = a ≼ b
