LongLeftRightArrow
==================

`WML link <https://reference.wolfram.com/language/ref/LongLeftRightArrow.html>`_


:code:`LongLeftRightArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⟷ :math:`y` ⟷ ...





>>> LongLeftRightArrow[x, y, z]
  = x ⟷ y ⟷ z
>>> a \[LongLeftRightArrow] b
  = a ⟷ b
