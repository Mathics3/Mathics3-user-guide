RightUpVectorBar
================

`WML link <https://reference.wolfram.com/language/ref/RightUpVectorBar.html>`_


:code:`RightUpVectorBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥔ :math:`y` ⥔ ...





>>> RightUpVectorBar[x, y, z]
    =

:math:`x \bar{\upharpoonright} y \bar{\upharpoonright} z`


>>> a \[RightUpVectorBar] b
    =

:math:`a \bar{\upharpoonright} b`
