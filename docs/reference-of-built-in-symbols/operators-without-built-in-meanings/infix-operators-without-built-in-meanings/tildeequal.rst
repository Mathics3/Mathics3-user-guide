TildeEqual
==========

`WML link <https://reference.wolfram.com/language/ref/TildeEqual.html>`_


:code:`TildeEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≃ :math:`y` ≃ ...





>>> TildeEqual[x, y, z]
    =

:math:`x \simeq y \simeq z`


>>> a \[TildeEqual] b
    =

:math:`a \simeq b`
