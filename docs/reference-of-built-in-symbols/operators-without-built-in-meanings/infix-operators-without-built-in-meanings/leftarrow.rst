LeftArrow
=========

`WML link <https://reference.wolfram.com/language/ref/LeftArrow.html>`_


:code:`LeftArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ← :math:`y` ← ...





>>> LeftArrow[x, y, z]

    =
:math:`x \leftarrow y \leftarrow z`


>>> a \[LeftArrow] b

    =
:math:`a \leftarrow b`
