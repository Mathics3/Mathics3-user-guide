LessGreater
===========

`WML link <https://reference.wolfram.com/language/ref/LessGreater.html>`_


:code:`LessGreater[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≶ :math:`y` ≶ ...





>>> LessGreater[x, y, z]
    =

:math:`x \lessgtr y \lessgtr z`


>>> a \[LessGreater] b
    =

:math:`a \lessgtr b`
