RightArrowBar
=============

`WML link <https://reference.wolfram.com/language/ref/RightArrowBar.html>`_


:code:`RightArrowBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇥ :math:`y` ⇥ ...





>>> RightArrowBar[x, y, z]

    =
:math:`x \rightarrow | y \rightarrow | z`


>>> a \[RightArrowBar] b

    =
:math:`a \rightarrow | b`
