DoubleDownArrow
===============

`WML link <https://reference.wolfram.com/language/ref/DoubleDownArrow.html>`_


:code:`DoubleDownArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇓ :math:`y` ⇓ ...





>>> DoubleDownArrow[x, y, z]
    =

:math:`x \Downarrow y \Downarrow z`


>>> a \[DoubleDownArrow] b
    =

:math:`a \Downarrow b`
