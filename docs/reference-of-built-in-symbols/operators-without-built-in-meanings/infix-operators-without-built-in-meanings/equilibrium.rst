Equilibrium
===========

`WML link <https://reference.wolfram.com/language/ref/Equilibrium.html>`_


:code:`Equilibrium[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇌ :math:`y` ⇌ ...





>>> Equilibrium[x, y, z]
    =

:math:`x \rightleftharpoons y \rightleftharpoons z`


>>> a \[Equilibrium] b
    =

:math:`a \rightleftharpoons b`
