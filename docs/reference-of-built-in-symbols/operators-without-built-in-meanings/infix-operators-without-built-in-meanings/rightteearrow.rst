RightTeeArrow
=============

`WML link <https://reference.wolfram.com/language/ref/RightTeeArrow.html>`_


:code:`RightTeeArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↦ :math:`y` ↦ ...





>>> RightTeeArrow[x, y, z]

    =
:math:`x \vdash y \vdash z`


>>> a \[RightTeeArrow] b

    =
:math:`a \vdash b`
