LeftArrowRightArrow
===================

`WML link <https://reference.wolfram.com/language/ref/LeftArrowRightArrow.html>`_


:code:`LeftArrowRightArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇆ :math:`y` ⇆ ...





>>> LeftArrowRightArrow[x, y, z]
  = x ⇆ y ⇆ z
>>> a \[LeftArrowRightArrow] b
  = a ⇆ b
