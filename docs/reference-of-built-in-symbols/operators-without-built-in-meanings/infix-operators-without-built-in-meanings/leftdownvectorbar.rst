LeftDownVectorBar
=================

`WML link <https://reference.wolfram.com/language/ref/LeftDownVectorBar.html>`_


:code:`LeftDownVectorBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥙ :math:`y` ⥙ ...





>>> LeftDownVectorBar[x, y, z]
  = x ⥙ y ⥙ z
>>> a \[LeftDownVectorBar] b
  = a ⥙ b
