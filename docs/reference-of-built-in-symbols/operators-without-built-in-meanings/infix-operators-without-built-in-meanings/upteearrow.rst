UpTeeArrow
==========

`WML link <https://reference.wolfram.com/language/ref/UpTeeArrow.html>`_


:code:`UpTeeArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↥ :math:`y` ↥ ...





>>> UpTeeArrow[x, y, z]
    =

:math:`x \underline{\uparrow} y \underline{\uparrow} z`


>>> a \[UpTeeArrow] b
    =

:math:`a \underline{\uparrow} b`
