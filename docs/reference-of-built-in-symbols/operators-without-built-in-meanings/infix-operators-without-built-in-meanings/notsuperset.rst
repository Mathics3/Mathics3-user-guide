NotSuperset
===========

`WML link <https://reference.wolfram.com/language/ref/NotSuperset.html>`_


:code:`NotSuperset[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊅ :math:`y` ⊅ ...





>>> NotSuperset[x, y, z]

    =
:math:`x \not{\supset} y \not{\supset} z`


>>> a \[NotSuperset] b

    =
:math:`a \not{\supset} b`
