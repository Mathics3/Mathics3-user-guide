SupersetEqual
=============

`WML link <https://reference.wolfram.com/language/ref/SupersetEqual.html>`_


:code:`SupersetEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊇ :math:`y` ⊇ ...





>>> SupersetEqual[x, y, z]
    =

:math:`x \supseteq y \supseteq z`


>>> a \[SupersetEqual] b
    =

:math:`a \supseteq b`
