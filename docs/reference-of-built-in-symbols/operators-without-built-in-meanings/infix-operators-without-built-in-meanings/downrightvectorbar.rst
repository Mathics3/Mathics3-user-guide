DownRightVectorBar
==================

`WML link <https://reference.wolfram.com/language/ref/DownRightVectorBar.html>`_


:code:`DownRightVectorBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥗ :math:`y` ⥗ ...





>>> DownRightVectorBar[x, y, z]

    =
:math:`x \rightharpoondown | y \rightharpoondown | z`


>>> a \[DownRightVectorBar] b

    =
:math:`a \rightharpoondown | b`
