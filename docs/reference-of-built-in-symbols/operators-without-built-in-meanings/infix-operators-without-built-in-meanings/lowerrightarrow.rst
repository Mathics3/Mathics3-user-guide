LowerRightArrow
===============

`WML link <https://reference.wolfram.com/language/ref/LowerRightArrow.html>`_


:code:`LowerRightArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↘ :math:`y` ↘ ...





>>> LowerRightArrow[x, y, z]

    =
:math:`x \searrow y \searrow z`


>>> a \[LowerRightArrow] b

    =
:math:`a \searrow b`
