NotGreaterFullEqual
===================

`WML link <https://reference.wolfram.com/language/ref/NotGreaterFullEqual.html>`_


:code:`NotGreaterFullEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≩ :math:`y` ≩ ...





>>> NotGreaterFullEqual[x, y, z]
  = x ≩ y ≩ z
>>> a \[NotGreaterFullEqual] b
  = a ≩ b
