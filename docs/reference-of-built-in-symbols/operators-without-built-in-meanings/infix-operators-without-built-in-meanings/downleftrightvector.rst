DownLeftRightVector
===================

`WML link <https://reference.wolfram.com/language/ref/DownLeftRightVector.html>`_


:code:`DownLeftRightVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥐ :math:`y` ⥐ ...





>>> DownLeftRightVector[x, y, z]
  = x ⥐ y ⥐ z
>>> a \[DownLeftRightVector] b
  = a ⥐ b
