SubsetEqual
===========

`WML link <https://reference.wolfram.com/language/ref/SubsetEqual.html>`_


:code:`SubsetEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊆ :math:`y` ⊆ ...





>>> SubsetEqual[x, y, z]
    =

:math:`x \subseteq y \subseteq z`


>>> a \[SubsetEqual] b
    =

:math:`a \subseteq b`
