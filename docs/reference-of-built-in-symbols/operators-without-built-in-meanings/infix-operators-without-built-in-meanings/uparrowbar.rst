UpArrowBar
==========

`WML link <https://reference.wolfram.com/language/ref/UpArrowBar.html>`_


:code:`UpArrowBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⤒ :math:`y` ⤒ ...





>>> UpArrowBar[x, y, z]

    =
:math:`x \bar{\uparrow} y \bar{\uparrow} z`


>>> a \[UpArrowBar] b

    =
:math:`a \bar{\uparrow} b`
