DownLeftVectorBar
=================

`WML link <https://reference.wolfram.com/language/ref/DownLeftVectorBar.html>`_


:code:`DownLeftVectorBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥖ :math:`y` ⥖ ...





>>> DownLeftVectorBar[x, y, z]
  = x ⥖ y ⥖ z
>>> a \[DownLeftVectorBar] b
  = a ⥖ b
