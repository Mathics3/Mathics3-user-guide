GreaterFullEqual
================

`WML link <https://reference.wolfram.com/language/ref/GreaterFullEqual.html>`_


:code:`GreaterFullEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≧ :math:`y` ≧ ...





>>> GreaterFullEqual[x, y, z]
    =

:math:`x \geqq y \geqq z`


>>> a \[GreaterFullEqual] b
    =

:math:`a \geqq b`
