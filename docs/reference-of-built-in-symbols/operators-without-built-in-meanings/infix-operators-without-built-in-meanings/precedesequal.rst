PrecedesEqual
=============

`WML link <https://reference.wolfram.com/language/ref/PrecedesEqual.html>`_


:code:`PrecedesEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⪯ :math:`y` ⪯ ...





>>> PrecedesEqual[x, y, z]

    =
:math:`x \preceq y \preceq z`


>>> a \[PrecedesEqual] b

    =
:math:`a \preceq b`
