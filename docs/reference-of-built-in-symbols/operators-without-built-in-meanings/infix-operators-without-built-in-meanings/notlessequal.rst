NotLessEqual
============

`WML link <https://reference.wolfram.com/language/ref/NotLessEqual.html>`_


:code:`NotLessEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≰ :math:`y` ≰ ...





>>> NotLessEqual[x, y, z]

    =
:math:`x \nleq y \nleq z`


>>> a \[NotLessEqual] b

    =
:math:`a \nleq b`
