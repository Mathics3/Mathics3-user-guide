DownRightTeeVector
==================

`WML link <https://reference.wolfram.com/language/ref/DownRightTeeVector.html>`_


:code:`DownRightTeeVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥟ :math:`y` ⥟ ...





>>> DownRightTeeVector[x, y, z]
  = x ⥟ y ⥟ z
>>> a \[DownRightTeeVector] b
  = a ⥟ b
