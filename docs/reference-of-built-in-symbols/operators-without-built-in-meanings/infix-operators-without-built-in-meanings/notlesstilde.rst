NotLessTilde
============

`WML link <https://reference.wolfram.com/language/ref/NotLessTilde.html>`_


:code:`NotLessTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≴ :math:`y` ≴ ...





>>> NotLessTilde[x, y, z]
    =

:math:`x \not{\lesssim} y \not{\lesssim} z`


>>> a \[NotLessTilde] b
    =

:math:`a \not{\lesssim} b`
