UpperLeftArrow
==============

`WML link <https://reference.wolfram.com/language/ref/UpperLeftArrow.html>`_


:code:`UpperLeftArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↖ :math:`y` ↖ ...





>>> UpperLeftArrow[x, y, z]

    =
:math:`x \nwarrow y \nwarrow z`


>>> a \[UpperLeftArrow] b

    =
:math:`a \nwarrow b`
