RightArrowLeftArrow
===================

`WML link <https://reference.wolfram.com/language/ref/RightArrowLeftArrow.html>`_


:code:`RightArrowLeftArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇄ :math:`y` ⇄ ...





>>> RightArrowLeftArrow[x, y, z]
    =

:math:`x \rightleftarrows y \rightleftarrows z`


>>> a \[RightArrowLeftArrow] b
    =

:math:`a \rightleftarrows b`
