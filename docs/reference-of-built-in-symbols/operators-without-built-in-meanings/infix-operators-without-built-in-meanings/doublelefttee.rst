DoubleLeftTee
=============

`WML link <https://reference.wolfram.com/language/ref/DoubleLeftTee.html>`_


:code:`DoubleLeftTee[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⫤ :math:`y` ⫤ ...





>>> DoubleLeftTee[x, y, z]

    =
:math:`x =| y =| z`


>>> a \[DoubleLeftTee] b

    =
:math:`a =| b`
