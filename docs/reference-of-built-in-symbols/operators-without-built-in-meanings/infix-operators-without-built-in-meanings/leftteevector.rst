LeftTeeVector
=============

`WML link <https://reference.wolfram.com/language/ref/LeftTeeVector.html>`_


:code:`LeftTeeVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥚ :math:`y` ⥚ ...





>>> LeftTeeVector[x, y, z]

    =
:math:`x \leftharpoonup | y \leftharpoonup | z`


>>> a \[LeftTeeVector] b

    =
:math:`a \leftharpoonup | b`
