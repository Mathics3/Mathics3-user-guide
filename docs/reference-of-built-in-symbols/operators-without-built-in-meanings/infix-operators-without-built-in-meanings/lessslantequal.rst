LessSlantEqual
==============

`WML link <https://reference.wolfram.com/language/ref/LessSlantEqual.html>`_


:code:`LessSlantEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⩽ :math:`y` ⩽ ...





>>> LessSlantEqual[x, y, z]

    =
:math:`x \leq y \leq z`


>>> a \[LessSlantEqual] b

    =
:math:`a \leq b`
