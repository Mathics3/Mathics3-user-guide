Perpendicular
=============

`WML link <https://reference.wolfram.com/language/ref/Perpendicular.html>`_


:code:`Perpendicular[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⟂ :math:`y` ⟂ ...





>>> Perpendicular[x, y, z]

    =
:math:`x \perp y \perp z`


>>> a \[Perpendicular] b

    =
:math:`a \perp b`
