LeftUpVector
============

`WML link <https://reference.wolfram.com/language/ref/LeftUpVector.html>`_


:code:`LeftUpVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↿ :math:`y` ↿ ...





>>> LeftUpVector[x, y, z]
    =

:math:`x \upharpoonleft y \upharpoonleft z`


>>> a \[LeftUpVector] b
    =

:math:`a \upharpoonleft b`
