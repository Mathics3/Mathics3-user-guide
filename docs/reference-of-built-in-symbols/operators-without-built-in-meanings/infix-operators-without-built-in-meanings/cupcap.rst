CupCap
======

`WML link <https://reference.wolfram.com/language/ref/CupCap.html>`_


:code:`CupCap[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≍ :math:`y` ≍ ...





>>> CupCap[x, y, z]
    =

:math:`x \stackrel{\smile}{\frown} y \stackrel{\smile}{\frown} z`


>>> a \[CupCap] b
    =

:math:`a \stackrel{\smile}{\frown} b`
