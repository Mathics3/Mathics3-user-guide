RightArrow
==========

`WML link <https://reference.wolfram.com/language/ref/RightArrow.html>`_


:code:`RightArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` → :math:`y` → ...





>>> RightArrow[x, y, z]
    =

:math:`x \rightarrow y \rightarrow z`


>>> a \[RightArrow] b
    =

:math:`a \rightarrow b`
