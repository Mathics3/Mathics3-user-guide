NotReverseElement
=================

`WML link <https://reference.wolfram.com/language/ref/NotReverseElement.html>`_


:code:`NotReverseElement[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ∌ :math:`y` ∌ ...





>>> NotReverseElement[x, y, z]
  = x ∌ y ∌ z
>>> a \[NotReverseElement] b
  = a ∌ b
