CircleTimes
===========

`WML link <https://reference.wolfram.com/language/ref/CircleTimes.html>`_


:code:`CircleTimes[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊗ :math:`y` ⊗ ...





>>> CircleTimes[x, y, z]
    =

:math:`x \otimes y \otimes z`


>>> a \[CircleTimes] b
    =

:math:`a \otimes b`
