Therefore
=========

`WML link <https://reference.wolfram.com/language/ref/Therefore.html>`_


:code:`Therefore[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ∴ :math:`y` ∴ ...





>>> Therefore[x, y, z]

    =
:math:`x \therefore y \therefore z`


>>> a \[Therefore] b

    =
:math:`a \therefore b`
