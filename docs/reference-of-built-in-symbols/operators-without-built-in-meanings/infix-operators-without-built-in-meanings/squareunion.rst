SquareUnion
===========

`WML link <https://reference.wolfram.com/language/ref/SquareUnion.html>`_


:code:`SquareUnion[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊔ :math:`y` ⊔ ...





>>> SquareUnion[x, y, z]

    =
:math:`x \sqcup y \sqcup z`


>>> a \[SquareUnion] b

    =
:math:`a \sqcup b`
