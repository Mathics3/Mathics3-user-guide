RightDownTeeVector
==================

`WML link <https://reference.wolfram.com/language/ref/RightDownTeeVector.html>`_


:code:`RightDownTeeVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥝ :math:`y` ⥝ ...





>>> RightDownTeeVector[x, y, z]
  = x ⥝ y ⥝ z
>>> a \[RightDownTeeVector] b
  = a ⥝ b
