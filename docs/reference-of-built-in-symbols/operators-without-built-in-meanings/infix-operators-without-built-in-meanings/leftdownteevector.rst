LeftDownTeeVector
=================

`WML link <https://reference.wolfram.com/language/ref/LeftDownTeeVector.html>`_


:code:`LeftDownTeeVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥡ :math:`y` ⥡ ...





>>> LeftDownTeeVector[x, y, z]
  = x ⥡ y ⥡ z
>>> a \[LeftDownTeeVector] b
  = a ⥡ b
