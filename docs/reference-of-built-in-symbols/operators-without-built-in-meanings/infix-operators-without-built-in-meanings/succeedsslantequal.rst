SucceedsSlantEqual
==================

`WML link <https://reference.wolfram.com/language/ref/SucceedsSlantEqual.html>`_


:code:`SucceedsSlantEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≽ :math:`y` ≽ ...





>>> SucceedsSlantEqual[x, y, z]
    =

:math:`x \succeq y \succeq z`


>>> a \[SucceedsSlantEqual] b
    =

:math:`a \succeq b`
