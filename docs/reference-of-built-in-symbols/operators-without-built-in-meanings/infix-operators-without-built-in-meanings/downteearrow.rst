DownTeeArrow
============

`WML link <https://reference.wolfram.com/language/ref/DownTeeArrow.html>`_


:code:`DownTeeArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↧ :math:`y` ↧ ...





>>> DownTeeArrow[x, y, z]

    =
:math:`x \bar{\downarrow} y \bar{\downarrow} z`


>>> a \[DownTeeArrow] b

    =
:math:`a \bar{\downarrow} b`
