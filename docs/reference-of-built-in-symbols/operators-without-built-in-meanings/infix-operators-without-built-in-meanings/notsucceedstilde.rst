NotSucceedsTilde
================

`WML link <https://reference.wolfram.com/language/ref/NotSucceedsTilde.html>`_


:code:`NotSucceedsTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⋩ :math:`y` ⋩ ...





>>> NotSucceedsTilde[x, y, z]
    =

:math:`x \not{\succsim} y \not{\succsim} z`


>>> a \[NotSucceedsTilde] b
    =

:math:`a \not{\succsim} b`
