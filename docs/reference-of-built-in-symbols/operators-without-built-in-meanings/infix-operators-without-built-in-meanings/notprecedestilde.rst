NotPrecedesTilde
================

`WML link <https://reference.wolfram.com/language/ref/NotPrecedesTilde.html>`_


:code:`NotPrecedesTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⋨ :math:`y` ⋨ ...





>>> NotPrecedesTilde[x, y, z]

    =
:math:`x \not{\precsim} y \not{\precsim} z`


>>> a \[NotPrecedesTilde] b

    =
:math:`a \not{\precsim} b`
