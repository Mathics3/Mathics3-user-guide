NotSquareSubsetEqual
====================

`WML link <https://reference.wolfram.com/language/ref/NotSquareSubsetEqual.html>`_


:code:`NotSquareSubsetEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⋢ :math:`y` ⋢ ...





>>> NotSquareSubsetEqual[x, y, z]

    =
:math:`x \not{\sqsubseteq} y \not{\sqsubseteq} z`


>>> a \[NotSquareSubsetEqual] b

    =
:math:`a \not{\sqsubseteq} b`
