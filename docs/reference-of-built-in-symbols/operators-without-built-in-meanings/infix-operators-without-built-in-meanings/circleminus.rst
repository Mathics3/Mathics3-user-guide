CircleMinus
===========

`WML link <https://reference.wolfram.com/language/ref/CircleMinus.html>`_


:code:`CircleMinus[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊖ :math:`y` ⊖ ...





>>> CircleMinus[x, y, z]
    =

:math:`x \ominus y \ominus z`


>>> a \[CircleMinus] b
    =

:math:`a \ominus b`
