LessTilde
=========

`WML link <https://reference.wolfram.com/language/ref/LessTilde.html>`_


:code:`LessTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≲ :math:`y` ≲ ...





>>> LessTilde[x, y, z]
    =

:math:`x \lesssim y \lesssim z`


>>> a \[LessTilde] b
    =

:math:`a \lesssim b`
