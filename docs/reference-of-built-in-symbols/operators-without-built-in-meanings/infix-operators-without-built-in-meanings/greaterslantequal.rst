GreaterSlantEqual
=================

`WML link <https://reference.wolfram.com/language/ref/GreaterSlantEqual.html>`_


:code:`GreaterSlantEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⩾ :math:`y` ⩾ ...





>>> GreaterSlantEqual[x, y, z]
    =

:math:`x \geq y \geq z`


>>> a \[GreaterSlantEqual] b
    =

:math:`a \geq b`
