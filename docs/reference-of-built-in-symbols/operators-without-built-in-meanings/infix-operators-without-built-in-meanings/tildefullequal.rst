TildeFullEqual
==============

`WML link <https://reference.wolfram.com/language/ref/TildeFullEqual.html>`_


:code:`TildeFullEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≅ :math:`y` ≅ ...





>>> TildeFullEqual[x, y, z]
    =

:math:`x \cong y \cong z`


>>> a \[TildeFullEqual] b
    =

:math:`a \cong b`
