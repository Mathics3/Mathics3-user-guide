GreaterTilde
============

`WML link <https://reference.wolfram.com/language/ref/GreaterTilde.html>`_


:code:`GreaterTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≳ :math:`y` ≳ ...





>>> GreaterTilde[x, y, z]
    =

:math:`x \gtrsim y \gtrsim z`


>>> a \[GreaterTilde] b
    =

:math:`a \gtrsim b`
