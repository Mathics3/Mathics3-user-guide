LongLeftArrow
=============

`WML link <https://reference.wolfram.com/language/ref/LongLeftArrow.html>`_


:code:`LongLeftArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⟵ :math:`y` ⟵ ...





>>> LongLeftArrow[x, y, z]
    =

:math:`x \longleftarrow y \longleftarrow z`


>>> a \[LongLeftArrow] b
    =

:math:`a \longleftarrow b`
