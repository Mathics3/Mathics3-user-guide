SquareSubsetEqual
=================

`WML link <https://reference.wolfram.com/language/ref/SquareSubsetEqual.html>`_


:code:`SquareSubsetEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊑ :math:`y` ⊑ ...





>>> SquareSubsetEqual[x, y, z]

    =
:math:`x \sqsubseteq y \sqsubseteq z`


>>> a \[SquareSubsetEqual] b

    =
:math:`a \sqsubseteq b`
