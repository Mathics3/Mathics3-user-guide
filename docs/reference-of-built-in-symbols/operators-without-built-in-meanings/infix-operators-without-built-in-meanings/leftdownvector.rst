LeftDownVector
==============

`WML link <https://reference.wolfram.com/language/ref/LeftDownVector.html>`_


:code:`LeftDownVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇃ :math:`y` ⇃ ...





>>> LeftDownVector[x, y, z]
    =

:math:`x \downharpoonleft y \downharpoonleft z`


>>> a \[LeftDownVector] b
    =

:math:`a \downharpoonleft b`
