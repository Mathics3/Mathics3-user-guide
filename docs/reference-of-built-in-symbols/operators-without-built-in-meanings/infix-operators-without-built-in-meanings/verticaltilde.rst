VerticalTilde
=============

`WML link <https://reference.wolfram.com/language/ref/VerticalTilde.html>`_


:code:`VerticalTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≀ :math:`y` ≀ ...





>>> VerticalTilde[x, y, z]

    =
:math:`x \wr y \wr z`


>>> a \[VerticalTilde] b

    =
:math:`a \wr b`
