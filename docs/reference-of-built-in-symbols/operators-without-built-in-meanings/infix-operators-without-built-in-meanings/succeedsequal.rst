SucceedsEqual
=============

`WML link <https://reference.wolfram.com/language/ref/SucceedsEqual.html>`_


:code:`SucceedsEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⪰ :math:`y` ⪰ ...





>>> SucceedsEqual[x, y, z]

    =
:math:`x \succeq y \succeq z`


>>> a \[SucceedsEqual] b

    =
:math:`a \succeq b`
