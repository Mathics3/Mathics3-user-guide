RightDownVectorBar
==================

`WML link <https://reference.wolfram.com/language/ref/RightDownVectorBar.html>`_


:code:`RightDownVectorBar[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥕ :math:`y` ⥕ ...





>>> RightDownVectorBar[x, y, z]
  = x ⥕ y ⥕ z
>>> a \[RightDownVectorBar] b
  = a ⥕ b
