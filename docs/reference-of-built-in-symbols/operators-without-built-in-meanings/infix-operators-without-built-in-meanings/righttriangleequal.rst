RightTriangleEqual
==================

`WML link <https://reference.wolfram.com/language/ref/RightTriangleEqual.html>`_


:code:`RightTriangleEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊵ :math:`y` ⊵ ...





>>> RightTriangleEqual[x, y, z]
  = x ⊵ y ⊵ z
>>> a \[RightTriangleEqual] b
  = a ⊵ b
