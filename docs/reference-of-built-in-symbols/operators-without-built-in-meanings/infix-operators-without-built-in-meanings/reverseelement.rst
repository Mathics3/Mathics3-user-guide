ReverseElement
==============

`WML link <https://reference.wolfram.com/language/ref/ReverseElement.html>`_


:code:`ReverseElement[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ∋ :math:`y` ∋ ...





>>> ReverseElement[x, y, z]
    =

:math:`x \ni y \ni z`


>>> a \[ReverseElement] b
    =

:math:`a \ni b`
