GreaterLess
===========

`WML link <https://reference.wolfram.com/language/ref/GreaterLess.html>`_


:code:`GreaterLess[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≷ :math:`y` ≷ ...





>>> GreaterLess[x, y, z]
    =

:math:`x \gtrless y \gtrless z`


>>> a \[GreaterLess] b
    =

:math:`a \gtrless b`
