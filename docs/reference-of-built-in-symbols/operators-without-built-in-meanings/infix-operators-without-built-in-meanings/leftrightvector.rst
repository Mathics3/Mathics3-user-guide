LeftRightVector
===============

`WML link <https://reference.wolfram.com/language/ref/LeftRightVector.html>`_


:code:`LeftRightVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥎ :math:`y` ⥎ ...





>>> LeftRightVector[x, y, z]

    =
:math:`x \leftharpoonup \rightharpoonup y \leftharpoonup \rightharpoonup z`


>>> a \[LeftRightVector] b

    =
:math:`a \leftharpoonup \rightharpoonup b`
