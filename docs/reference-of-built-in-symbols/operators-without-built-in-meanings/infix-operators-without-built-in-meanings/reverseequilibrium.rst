ReverseEquilibrium
==================

`WML link <https://reference.wolfram.com/language/ref/ReverseEquilibrium.html>`_


:code:`ReverseEquilibrium[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇋ :math:`y` ⇋ ...





>>> ReverseEquilibrium[x, y, z]
  = x ⇋ y ⇋ z
>>> a \[ReverseEquilibrium] b
  = a ⇋ b
