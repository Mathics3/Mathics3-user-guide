Proportional
============

`WML link <https://reference.wolfram.com/language/ref/Proportional.html>`_


:code:`Proportional[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ∝ :math:`y` ∝ ...





>>> Proportional[x, y, z]
    =

:math:`x \propto y \propto z`


>>> a \[Proportional] b
    =

:math:`a \propto b`
