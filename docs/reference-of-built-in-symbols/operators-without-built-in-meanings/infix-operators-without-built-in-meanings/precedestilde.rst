PrecedesTilde
=============

`WML link <https://reference.wolfram.com/language/ref/PrecedesTilde.html>`_


:code:`PrecedesTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≾ :math:`y` ≾ ...





>>> PrecedesTilde[x, y, z]

    =
:math:`x \precsim y \precsim z`


>>> a \[PrecedesTilde] b

    =
:math:`a \precsim b`
