ShortUpArrow
============

`WML link <https://reference.wolfram.com/language/ref/ShortUpArrow.html>`_


:code:`ShortUpArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↑ :math:`y` ↑ ...





>>> ShortUpArrow[x, y, z]
    =

:math:`x \uparrow y \uparrow z`


>>> a \[ShortUpArrow] b
    =

:math:`a \uparrow b`
