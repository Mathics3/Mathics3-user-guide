SmallCircle
===========

`WML link <https://reference.wolfram.com/language/ref/SmallCircle.html>`_


:code:`SmallCircle[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ∘ :math:`y` ∘ ...





>>> SmallCircle[x, y, z]

    =
:math:`x \circ y \circ z`


>>> a \[SmallCircle] b

    =
:math:`a \circ b`
