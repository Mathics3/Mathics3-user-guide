LeftTriangleEqual
=================

`WML link <https://reference.wolfram.com/language/ref/LeftTriangleEqual.html>`_


:code:`LeftTriangleEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊴ :math:`y` ⊴ ...





>>> LeftTriangleEqual[x, y, z]
  = x ⊴ y ⊴ z
>>> a \[LeftTriangleEqual] b
  = a ⊴ b
