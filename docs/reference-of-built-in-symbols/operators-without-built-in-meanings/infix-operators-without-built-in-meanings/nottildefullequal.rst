NotTildeFullEqual
=================

`WML link <https://reference.wolfram.com/language/ref/NotTildeFullEqual.html>`_


:code:`NotTildeFullEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≇ :math:`y` ≇ ...





>>> NotTildeFullEqual[x, y, z]
    =

:math:`x \ncong y \ncong z`


>>> a \[NotTildeFullEqual] b
    =

:math:`a \ncong b`
