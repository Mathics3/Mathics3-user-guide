DoubleRightTee
==============

`WML link <https://reference.wolfram.com/language/ref/DoubleRightTee.html>`_


:code:`DoubleRightTee[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊨ :math:`y` ⊨ ...





>>> DoubleRightTee[x, y, z]
    =

:math:`x \vDash y \vDash z`


>>> a \[DoubleRightTee] b
    =

:math:`a \vDash b`
