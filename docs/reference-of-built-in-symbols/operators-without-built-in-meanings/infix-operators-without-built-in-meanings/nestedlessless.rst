NestedLessLess
==============

`WML link <https://reference.wolfram.com/language/ref/NestedLessLess.html>`_


:code:`NestedLessLess[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⪡ :math:`y` ⪡ ...





>>> NestedLessLess[x, y, z]
    =

:math:`x \ll y \ll z`


>>> a \[NestedLessLess] b
    =

:math:`a \ll b`
