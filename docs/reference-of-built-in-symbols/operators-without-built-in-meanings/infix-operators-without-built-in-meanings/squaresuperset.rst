SquareSuperset
==============

`WML link <https://reference.wolfram.com/language/ref/SquareSuperset.html>`_


:code:`SquareSuperset[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊐ :math:`y` ⊐ ...





>>> SquareSuperset[x, y, z]
    =

:math:`x \sqsupset y \sqsupset z`


>>> a \[SquareSuperset] b
    =

:math:`a \sqsupset b`
