NotGreaterEqual
===============

`WML link <https://reference.wolfram.com/language/ref/NotGreaterEqual.html>`_


:code:`NotGreaterEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≱ :math:`y` ≱ ...





>>> NotGreaterEqual[x, y, z]
    =

:math:`x \ngeq y \ngeq z`


>>> a \[NotGreaterEqual] b
    =

:math:`a \ngeq b`
