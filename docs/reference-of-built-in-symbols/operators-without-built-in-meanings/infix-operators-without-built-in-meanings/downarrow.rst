DownArrow
=========

`WML link <https://reference.wolfram.com/language/ref/DownArrow.html>`_


:code:`DownArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↓ :math:`y` ↓ ...





>>> DownArrow[x, y, z]

    =
:math:`x \downarrow y \downarrow z`


>>> a \[DownArrow] b

    =
:math:`a \downarrow b`
