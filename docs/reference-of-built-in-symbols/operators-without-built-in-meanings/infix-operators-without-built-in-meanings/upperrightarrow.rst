UpperRightArrow
===============

`WML link <https://reference.wolfram.com/language/ref/UpperRightArrow.html>`_


:code:`UpperRightArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↗ :math:`y` ↗ ...





>>> UpperRightArrow[x, y, z]
    =

:math:`x \nearrow y \nearrow z`


>>> a \[UpperRightArrow] b
    =

:math:`a \nearrow b`
