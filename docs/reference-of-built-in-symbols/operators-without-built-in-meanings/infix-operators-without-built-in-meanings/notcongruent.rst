NotCongruent
============

`WML link <https://reference.wolfram.com/language/ref/NotCongruent.html>`_


:code:`NotCongruent[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≢ :math:`y` ≢ ...





>>> NotCongruent[x, y, z]

    =
:math:`x \not{\equiv} y \not{\equiv} z`


>>> a \[NotCongruent] b

    =
:math:`a \not{\equiv} b`
