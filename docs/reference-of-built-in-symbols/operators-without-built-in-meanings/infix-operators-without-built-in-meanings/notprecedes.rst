NotPrecedes
===========

`WML link <https://reference.wolfram.com/language/ref/NotPrecedes.html>`_


:code:`NotPrecedes[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊀ :math:`y` ⊀ ...





>>> NotPrecedes[x, y, z]
    =

:math:`x \nprec y \nprec z`


>>> a \[NotPrecedes] b
    =

:math:`a \nprec b`
