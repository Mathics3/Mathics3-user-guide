ShortLeftArrow
==============

`WML link <https://reference.wolfram.com/language/ref/ShortLeftArrow.html>`_


:code:`ShortLeftArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ← :math:`y` ← ...





>>> ShortLeftArrow[x, y, z]

    =
:math:`x \leftarrow y \leftarrow z`


>>> a \[ShortLeftArrow] b

    =
:math:`a \leftarrow b`
