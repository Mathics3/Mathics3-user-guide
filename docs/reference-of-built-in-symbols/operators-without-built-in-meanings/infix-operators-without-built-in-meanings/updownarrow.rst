UpDownArrow
===========

`WML link <https://reference.wolfram.com/language/ref/UpDownArrow.html>`_


:code:`UpDownArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↕ :math:`y` ↕ ...





>>> UpDownArrow[x, y, z]
    =

:math:`x \updownarrow y \updownarrow z`


>>> a \[UpDownArrow] b
    =

:math:`a \updownarrow b`
