RightUpDownVector
=================

`WML link <https://reference.wolfram.com/language/ref/RightUpDownVector.html>`_


:code:`RightUpDownVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⥏ :math:`y` ⥏ ...





>>> RightUpDownVector[x, y, z]
  = x ⥏ y ⥏ z
>>> a \[RightUpDownVector] b
  = a ⥏ b
