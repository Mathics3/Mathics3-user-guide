NotSucceedsSlantEqual
=====================

`WML link <https://reference.wolfram.com/language/ref/NotSucceedsSlantEqual.html>`_


:code:`NotSucceedsSlantEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⋡ :math:`y` ⋡ ...





>>> NotSucceedsSlantEqual[x, y, z]
    =

:math:`x \not{\succeq} y \not{\succeq} z`


>>> a \[NotSucceedsSlantEqual] b
    =

:math:`a \not{\succeq} b`
