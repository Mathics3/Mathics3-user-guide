SuchThat
========

`WML link <https://reference.wolfram.com/language/ref/SuchThat.html>`_


:code:`SuchThat[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ∍ :math:`y` ∍ ...





>>> SuchThat[x, y, z]

    =
:math:`x \backepsilon y \backepsilon z`


>>> a \[SuchThat] b

    =
:math:`a \backepsilon b`
