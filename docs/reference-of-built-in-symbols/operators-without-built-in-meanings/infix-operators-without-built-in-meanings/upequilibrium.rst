UpEquilibrium
=============

`WML link <https://reference.wolfram.com/language/ref/UpEquilibrium.html>`_


:code:`UpEquilibrium[:math:`x`, :math:`y`, ...]`
    displays :math:`x` тео :math:`y` тео ...





>>> UpEquilibrium[x, y, z]

    =
:math:`x \upharpoonleft \downharpoonright y \upharpoonleft \downharpoonright z`


>>> a \[UpEquilibrium] b

    =
:math:`a \upharpoonleft \downharpoonright b`
