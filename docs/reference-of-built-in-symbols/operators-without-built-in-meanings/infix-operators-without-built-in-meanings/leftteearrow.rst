LeftTeeArrow
============

`WML link <https://reference.wolfram.com/language/ref/LeftTeeArrow.html>`_


:code:`LeftTeeArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↤ :math:`y` ↤ ...





>>> LeftTeeArrow[x, y, z]
    =

:math:`x \mapsfrom y \mapsfrom z`


>>> a \[LeftTeeArrow] b
    =

:math:`a \mapsfrom b`
