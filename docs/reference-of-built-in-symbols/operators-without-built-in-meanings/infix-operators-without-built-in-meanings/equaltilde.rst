EqualTilde
==========

`WML link <https://reference.wolfram.com/language/ref/EqualTilde.html>`_


:code:`EqualTilde[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≂ :math:`y` ≂ ...





>>> EqualTilde[x, y, z]

    =
:math:`x \eqsim y \eqsim z`


>>> a \[EqualTilde] b

    =
:math:`a \eqsim b`
