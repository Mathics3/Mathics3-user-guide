NotTildeEqual
=============

`WML link <https://reference.wolfram.com/language/ref/NotTildeEqual.html>`_


:code:`NotTildeEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ≄ :math:`y` ≄ ...





>>> NotTildeEqual[x, y, z]
    =

:math:`x \not{\simeq} y \not{\simeq} z`


>>> a \[NotTildeEqual] b
    =

:math:`a \not{\simeq} b`
