NotSupersetEqual
================

`WML link <https://reference.wolfram.com/language/ref/NotSupersetEqual.html>`_


:code:`NotSupersetEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊉ :math:`y` ⊉ ...





>>> NotSupersetEqual[x, y, z]

    =
:math:`x \nsupseteq y \nsupseteq z`


>>> a \[NotSupersetEqual] b

    =
:math:`a \nsupseteq b`
