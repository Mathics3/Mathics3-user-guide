SquareSupersetEqual
===================

`WML link <https://reference.wolfram.com/language/ref/SquareSupersetEqual.html>`_


:code:`SquareSupersetEqual[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊒ :math:`y` ⊒ ...





>>> SquareSupersetEqual[x, y, z]

    =
:math:`x \sqsupseteq y \sqsupseteq z`


>>> a \[SquareSupersetEqual] b

    =
:math:`a \sqsupseteq b`
