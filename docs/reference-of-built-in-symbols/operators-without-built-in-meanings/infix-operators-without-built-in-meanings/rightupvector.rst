RightUpVector
=============

`WML link <https://reference.wolfram.com/language/ref/RightUpVector.html>`_


:code:`RightUpVector[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ↾ :math:`y` ↾ ...





>>> RightUpVector[x, y, z]
    =

:math:`x \upharpoonright y \upharpoonright z`


>>> a \[RightUpVector] b
    =

:math:`a \upharpoonright b`
