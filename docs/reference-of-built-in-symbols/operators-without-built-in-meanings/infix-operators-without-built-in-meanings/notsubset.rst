NotSubset
=========

`WML link <https://reference.wolfram.com/language/ref/NotSubset.html>`_


:code:`NotSubset[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⊄ :math:`y` ⊄ ...





>>> NotSubset[x, y, z]
    =

:math:`x \not{\subset} y \not{\subset} z`


>>> a \[NotSubset] b
    =

:math:`a \not{\subset} b`
