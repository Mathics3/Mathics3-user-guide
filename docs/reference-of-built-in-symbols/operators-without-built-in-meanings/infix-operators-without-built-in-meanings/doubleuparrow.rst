DoubleUpArrow
=============

`WML link <https://reference.wolfram.com/language/ref/DoubleUpArrow.html>`_


:code:`DoubleUpArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇑ :math:`y` ⇑ ...





>>> DoubleUpArrow[x, y, z]

    =
:math:`x \Uparrow y \Uparrow z`


>>> a \[DoubleUpArrow] b

    =
:math:`a \Uparrow b`
