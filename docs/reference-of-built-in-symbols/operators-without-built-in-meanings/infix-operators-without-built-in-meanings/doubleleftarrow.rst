DoubleLeftArrow
===============

`WML link <https://reference.wolfram.com/language/ref/DoubleLeftArrow.html>`_


:code:`DoubleLeftArrow[:math:`x`, :math:`y`, ...]`
    displays :math:`x` ⇐ :math:`y` ⇐ ...





>>> DoubleLeftArrow[x, y, z]

    =
:math:`x \Leftarrow y \Leftarrow z`


>>> a \[DoubleLeftArrow] b

    =
:math:`a \Leftarrow b`
