$HistoryLength
==============

`WMA <https://reference.wolfram.com/language/ref/$HistoryLength>`_

:code:`$HistoryLength`
    specifies the maximum number of :code:`In`  and :code:`Out`  entries.





>>> $HistoryLength

    =
:math:`100`


>>> $HistoryLength = 1;


>>> 42

    =
:math:`42`


>>> %

    =
:math:`42`


>>> %%

    =
:math:`\text{\%3}`


>>> $HistoryLength = 0;


>>> 42

    =
:math:`42`


>>> %

    =
:math:`\text{\%7}`
