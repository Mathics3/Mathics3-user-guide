$HistoryLength
==============

`WMA <https://reference.wolfram.com/language/ref/$HistoryLength>`_

:code:`$HistoryLength`
    specifies the maximum number of :code:`In`  and :code:`Out`  entries.





>>> $HistoryLength
  = 100
>>> $HistoryLength = 1;

>>> 42
  = 42
>>> %
  = 42
>>> %%
  = %3
>>> $HistoryLength = 0;

>>> 42
  = 42
>>> %
  = %7
