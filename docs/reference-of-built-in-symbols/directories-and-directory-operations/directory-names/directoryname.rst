DirectoryName
=============

`WMA link <https://reference.wolfram.com/language/ref/DirectoryName.html>`_


:code:`DirectoryName` [":math:`name`"]
    extracts the directory name from a filename.





>>> DirectoryName["a/b/c"]
    =

:math:`\text{a/b}`


>>> DirectoryName["a/b/c", 2]
    =

:math:`\text{a}`
