FileExtension
=============

`WMA link <https://reference.wolfram.com/language/ref/FileExtension.html>`_


:code:`FileExtension` [":math:`file`"]
    gives the extension for the specified file name.





>>> FileExtension["file.txt"]

    =
:math:`\text{txt}`


>>> FileExtension["file.tar.gz"]

    =
:math:`\text{gz}`
