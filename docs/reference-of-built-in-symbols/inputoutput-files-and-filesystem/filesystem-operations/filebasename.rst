FileBaseName
============

`WMA link <https://reference.wolfram.com/language/ref/FileBaseName.html>`_


:code:`FileBaseName` [":math:`file`"]
    gives the base name for the specified file name.





>>> FileBaseName["file.txt"]
    =

:math:`\text{file}`


>>> FileBaseName["file.tar.gz"]
    =

:math:`\text{file.tar}`
