Mean
====

`WMA link <https://reference.wolfram.com/language/ref/Mean.html>`_


:code:`Mean` [:math:`list`]
    returns the statistical mean of :math:`list`.





>>> Mean[{26, 64, 36}]

    =
:math:`42`


>>> Mean[{1, 1, 2, 3, 5, 8}]

    =
:math:`\frac{10}{3}`


>>> Mean[{a, b}]

    =
:math:`\frac{a+b}{2}`
