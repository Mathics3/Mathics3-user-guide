SymbolQ
=======

`WMA link <https://reference.wolfram.com/language/ref/SymbolName.html>`_

:code:`SymbolQ` [:math:`x`]
    is :code:`True`  if :math:`x` is a symbol, or :code:`False`  otherwise.





>>> SymbolQ[a]
    =

:math:`\text{True}`


>>> SymbolQ[1]
    =

:math:`\text{False}`


>>> SymbolQ[a + b]
    =

:math:`\text{False}`
