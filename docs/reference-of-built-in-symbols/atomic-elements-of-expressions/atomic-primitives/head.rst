Head
====

`WMA link <https://reference.wolfram.com/language/ref/Head.html>`_


:code:`Head` [:math:`expr`]
    returns the head of the expression or atom :math:`expr`.





>>> Head[a * b]
    =

:math:`\text{Times}`


>>> Head[6]
    =

:math:`\text{Integer}`


>>> Head[x]
    =

:math:`\text{Symbol}`
