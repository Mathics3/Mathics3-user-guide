MachinePrecision
================

`WMA link <https://reference.wolfram.com/language/ref/MachinePrecision.html>`_

:code:`MachinePrecision`
    represents the precision of machine precision numbers.





>>> N[MachinePrecision]
  = 15.9546
>>> N[MachinePrecision, 30]
  = 15.9545897701910033463281614204
