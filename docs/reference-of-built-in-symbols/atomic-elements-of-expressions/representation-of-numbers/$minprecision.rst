$MinPrecision
=============

`WMA link <https://reference.wolfram.com/language/ref/$MinPrecision.html>`_


:code:`$MinPrecision`
    represents the minimum number of digits of precision permitted in abitrary-precision numbers.





>>> $MinPrecision
  = 0
>>> $MinPrecision = 10;

>>> N[Pi, 9]
  = 3.141592654
