Maximize
========

`WMA link <https://reference.wolfram.com/language/ref/Maximize.html>`_


:code:`Maximize` [:math:`f`, :math:`x`]
    compute the maximum of :math:`f` respect :math:`x` that change between       :math:`a` and :math:`b`.





>>> Maximize[-2 x^2 - 3 x + 5, x]
  = {{49 / 8, {x -> -3 / 4}}}
