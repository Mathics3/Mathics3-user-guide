Minimize
========

`WMA link <https://reference.wolfram.com/language/ref/Minimize.html>`_


:code:`Minimize` [:math:`f`, :math:`x`]
    compute the minimum of :math:`f` respect :math:`x` that change between         :math:`a` and :math:`b`.





>>> Minimize[2 x^2 - 3 x + 5, x]
  = {{31 / 8, {x -> 3 / 4}}}
